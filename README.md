# 🍕 FastAPI Food Delivery Backend

This project is a complete backend system built using FastAPI as part of my internship at Innomatics Research Labs.

## 🚀 Features

- REST API development using FastAPI
- Pydantic validation for request data
- CRUD operations for menu management
- Cart system with checkout workflow
- Multi-step order processing (Cart → Checkout → Order)
- Search functionality (case-insensitive)
- Sorting (price, name, category)
- Pagination support
- Combined browse endpoint
- API testing using Swagger UI

## 📂 Project Structure

- main.py → Core backend logic
- requirements.txt → Dependencies
- screenshots/ → API testing proof
